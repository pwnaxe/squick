# Getting Support

## Community support (free)

For questions, bug reports, and feature requests:

1. Search the [existing issues](https://github.com/pwnaxe/squick/issues)
   in case someone has already reported it.
2. If not, [open a new issue](https://github.com/pwnaxe/squick/issues/new/choose)
   using the appropriate template.

Community issues are triaged on a best-effort basis by the
maintainers.

Do not report security issues in the public tracker. Follow the
process in [SECURITY.md](SECURITY.md).
